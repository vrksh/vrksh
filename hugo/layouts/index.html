Install

curl -sSL https://vrk.sh/install.sh | sh

Works where agents work.

vrksh ships a discovery-only MCP server. Add it to Claude Code and every tool becomes visible to your agent. Tools run as shell commands - the MCP layer handles discovery, not execution. 

{
  "mcpServers": {
    "vrksh": {
      "command": "vrk",
      "args": ["mcp"]
    }
  }
}

Or use vrk --manifest for JSON tool discovery without MCP.
